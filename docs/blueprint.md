# **App Name**: Freshoz-fresh

## Core Features:

- Product Browsing and Smart Search: Firestore collections & indexing. AI tool suggests related items based on search history to increase order value.
- Real-time Order Tracking: Rider location updates & GeoFirestore
- Rider Assignment: Cloud Functions / geohash queries
- Inventory Management: Admin manages product stock, prices, and availability. products collection & transactions.
- Promotional Offers: promo_codes collection + Cloud Functions
- Earnings Summary: riders collection / transactions subcollection. Riders can view a summary of their earnings and shift logs.

## Style Guidelines:

- Primary color: Emerald green (#27AE60), symbolizing freshness and reliability.
- Background color: Light gray (#F7F7F7), for a clean and modern feel.
- Accent color: Amber (#F39C12), used for highlighting promotions and calls to action.
- Font pairing: 'Poppins' (sans-serif) for headlines and 'Inter' (sans-serif) for body text.
- Use a set of simple and clear icons to represent product categories and actions.
- Mobile-first design with accessible layouts for easy navigation on smaller screens.
- Subtle animations on product transitions and loading states to enhance user experience.
- Use a modern glassmorphism effect for cards and modals.
- Use neon lighting effect for primary action buttons (Add to Cart, Checkout). Keep it subtle, elegant, and mobile-friendly.
